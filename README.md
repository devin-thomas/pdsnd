# US Bikeshare Explorer

## Date created
June 16, 2026

## Description
US Bikeshare Explorer is a command-line Python project for reviewing bikeshare
trip data. The app is based on Udacity's starter project and is being prepared
in this repository as part of the Git and GitHub workflow assignment.

## Installation
1. Clone the repository to your local machine.
2. Create or activate a Python environment with `pandas` and `numpy`
   installed.
3. Place the required CSV data files in the project folder if you want to
   analyze cities beyond the ignored sample data file.

## Requirements
- Python 3
- pandas
- numpy

## Usage
Run the script from the project directory:

```bash
python bikeshare.py
```

The program prompts for a city, month, and day filter before printing travel
time, station, trip duration, and user statistics.
